# GGJ2025 #

A narrative game for Global Game Jam 2025.

### How to Play ###

Go to https://ehgoodenough.github.io/ggj2025/main for the latest build of the game!

### How to Build ###

Have node v18+ and npm v8+.

```
node --version
npm --version
```

Start the server.

```
npm install
npm start
```
